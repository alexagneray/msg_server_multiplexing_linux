### Nommage selon la portée
|préfixe|description|
|-----|-----|
|g_|globale|
|s_|statique|

|préfixe|description|
|-----|-----|

|c_|char|
|i_|int|
|u_|unsigned|
|l_|long|
|ul_|unsigned long|
|f_|float|
|d_|double|
|cnt_|nombre (count ou size)|

|psz_|char *pszList talbeau de char zero terminated|
|w_|wchar, caractère de grande taille|

|pi_|int*|
